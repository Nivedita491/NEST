# NEST Platform — Ecosystem Hub (Volunteer Project)

A full-stack MERN-based platform designed to unify startups, mentors, students, and administrators into a single digital ecosystem.

This project transforms a fragmented college ecosystem into an integrated, scalable platform for collaboration, discovery, and growth.

---

## 🌐 Overview
NEST (National Entrepreneurship Ecosystem Support Tool) is built as a dynamic platform to:
- Connect startups with mentors  
- Enable students to discover opportunities  
- Centralize events, resources, and ecosystem activities  

---

## 🚀 Features

- 📊 **Startup Directory** — Search and filter startups by domain and stage  
- 🧑‍🏫 **Mentor Directory** — Discover mentors with expertise and ratings  
- 📅 **Events Module** — Manage hackathons, seminars, and registrations  
- 🏢 **Resource Booking** — Book infrastructure and mentorship sessions  
- 🤖 **AI Matchmaking** — Suggest best mentor-startup matches  
- 🔐 **Role-Based Access** — Student, Founder, Mentor, Admin  
- 📱 **Responsive UI** — Mobile-first design using Tailwind CSS  
- ⚡ **PWA Support** — Installable web app with offline capabilities  

---
